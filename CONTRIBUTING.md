# 🌟 How to Contribute

## 1️⃣ Fork & Clone

```bash
git clone https://github.com/YOUR_USERNAME/OpenStream.git
```

---

## 2️⃣ Create Branch

```bash
git checkout -b feature/your-feature
```

Examples:
```bash
feature/player-ui
fix/navigation-bug
```

---

## 3️⃣ Make Changes

Please follow:

- Clean Architecture
- MVVM pattern
- Kotlin best practices
- Compose UI standards

---

## 4️⃣ Test Project

```bash
./gradlew assembleDebug
```

Ensure:
- No crashes
- Build succeeds
- Features work correctly

---

## 5️⃣ Commit Changes

```bash
git commit -m "feat: add new feature"
```

Examples:
```bash
fix: resolve playback crash
ui: improve onboarding screen
perf: optimize loading
```

---

## 6️⃣ Push & Open PR

```bash
git push origin your-branch-name
```

Then create a Pull Request on GitHub.

---

# ⚠ Guidelines

Please avoid:
- Uploading secrets/API keys
- Committing build folders
- Large unrelated changes

---
