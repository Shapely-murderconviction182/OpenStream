# AI Policy for OpenStream

## AI-Assisted Development
AI tools may assist with code suggestions, documentation, and bug identification.

## Guidelines
- All AI-generated code must be reviewed by a human
- AI-generated code must pass all tests before merging
- Security-sensitive code requires additional human review
- Disclose AI tool usage in PRs

## Prohibited
- Auto-merging AI-generated PRs without review
- Using AI to bypass security reviews
